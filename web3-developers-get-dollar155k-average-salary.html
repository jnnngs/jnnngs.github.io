Web3 Developers get $155k average salary

Do you want to get FAST into the industry? Build Projects. Make your Web3 Portfolio stand out with these 6 projects.

Create your own Token

Follow the tutorial and deploy your unique cryptocurrency.

Develop more than a simple ERC20 Token but also BEP20 on Binance Chain.

Add features:

  • Burnable
  • Mintable
  • Pausable
  • Deflation
  • Liquidity Fee

Mint your NFT

Non-Fungible Tokens are usually ERC721 Tokens.

Learn how to create your own NFT collection and upload it to OpenSea.

Dig into this tutorial to deploy an NFT project.

Build a dApp

Learn the fundamentals by building a decentralized App.

Start with a simple Smart Contract (backend) in Solidity.

Then a JavaScript front-end, all linked with web3.js

Go step by step using this tutorial.

Create your NFT Marketplace

A great way to build a Full Stack web3 app and explore many Ethereum concepts is to develop a NFT Marketplace.

Thanks to @dabit3 for this amazing guide.

Build a Decentralized Exchange (DEX)

This is an important DeFi project.

A DEX is like Coinbase or Binance but peer-to-peer. You trade crypto without an intermediary.

Fork PancakeSwap. Make your own UI and additional features.

Create a DAO

Decentralized Autonomous Organizations are Web3 communities where the power is bottom-up.

All the members get a share of the DAO (Tokens) and vote for the rules.

Go through this tutorial.
